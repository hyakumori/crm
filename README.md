## Requirements

- yarn >= 1.22.5

## Installation

1. Copy `.env.example` to `.env` and fill necessary variables.

```
cp .env.example .env
```

2. Install yarn dependencies.

```
yarn install
```

## Usage

Compile the app and launch with hot-reload for development with:

```
yarn serve
```

Or build for production with:

```
yarn build
```
